# Hi, I'm S.S. (SS-Embedded-Engineer) 👋

2017年より **組み込みエンジニア** として、産業用制御機器（通信・IOユニット）の垂直立ち上げから保守まで一貫して携わっています。
実務における設計判断のプロセスや、ハード・ソフトの境界領域における実装ノウハウを技術ログとして整理しています。

### 🌟 Summary of Experience
- **低レイヤ一貫開発:** 回路設計(CR8000等)、FPGA(VHDL/Verilog)、FW(C/ARM系)を跨いだ統合的な設計・デバッグ。
- **品質改善:** 独自のエラーログ解析機能の実装により、現場不具合の特定工数を大幅削減。迅速な原因究明と対策のサイクルを確立し、不具合発生率の抑制（月5件→0.5件）を実現。
- **自動化推進:** Python/VBAを用いたテスト・製造検査ツールの構築により、検証作業の自動化・高効率化を牽引。
- **知財意識:** 年2件ペースで特許明細作成に携わり、技術の独自性と保護を意識した設計を実践。

### 🛠️ Technical Stack
- **FPGA:** VHDL / Verilog (Quartus, Lattice Diamond, Vivado, Efinity)
- **FW:** C言語 (ARM Cortex-M / STM32, IAR)
- **Test / Automation:** Python / VBA / bat / Git
- **Circuit Design:** CR5000 / CR8000
